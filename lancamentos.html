ℹ️ Como funciona:
CARTÃO: A data de compra fica igual em todas as parcelas. Só a competência (mês da fatura) muda.
💡 Dicas:
• Use valores POSITIVOS para receitas (ex: 5000)
• Use valores NEGATIVOS para despesas (ex: -500)
• Data: Quando a transação foi realizada (compra, serviço, etc)
• Vencimento: Quando vai pagar/receber
• CARTÃO: Vencimento = Mês da fatura (11/2025)
• OUTROS: Vencimento = Data completa de pagamento (15/11/2025)
• Se escolher parcelas, o sistema cria automaticamente todos os vencimentos
